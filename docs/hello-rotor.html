Creating your first program

Now that the Rotor toolchain is installed and ready to go, let's create our first program! This document will help you make a program that prints "Rotor is spinning!" to the console and exits with a success code.

To create a new program, open your terminal in the desired location and run the command: 

rotor new my_first_project
Tip: If Rotor says Fatal: Permission denied, try running your terminal with administrator privileges.
On Windows, right-click your terminal and select Run as administrator.
On Unix-based systems, prefix your command with sudo (e.g., sudo rotor new my_first_project).
