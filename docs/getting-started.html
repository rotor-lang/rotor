Getting started with the rotor toolchain

So you want to start using the rotor toolchain? Well first of all, thank you for trying it out! Even if it's just for a little bit, I hope you have fun while using it! So let's get you all set up and ready to go to start making some awesome programs!

Installation

Currently, the rotor toolchain is still in development, so some things may not work as expected. But don't worry, we are working hard to make it better!

Windows

Windows users can download the pre-built binaries from the latest release.

  1. Download the latest Windows binaries from the latest release.
  2. Extract the contents to a folder of your choice.
  3. Add the folder to your PATH environment variable to use rotor from any Command Prompt.
  4. Open a new Command Prompt window and run rotor --version to verify.
Tip: Restart your PC after modifying the PATH variable for changes to take effect.

macOS

macOS users can install rotor using Homebrew:

brew install rotor
Tip: If you don't have Homebrew installed, check out brew.sh for setup instructions.

Linux

Linux users can download the .tar.gz archive from the latest release:

  1. Download the latest .tar.gz from the latest release.
  2. Extract it using tar -xzf rotor-version.tar.gz.
  3. Move the rotor binary to a directory in your PATH, like /usr/local/bin.
  4. Verify installation by running rotor --version.
Tip: Use chmod +x rotor if you get permission errors running the binary.
